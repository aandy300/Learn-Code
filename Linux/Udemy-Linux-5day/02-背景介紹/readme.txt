#什麼是Linux?
- 操作系統(以Unix為目標開發) 軟硬體的中介
- 內核 Kernel (Kernel可吃 Core 不可吃 )
- 開源軟體 FOSS (Free/Open Source Softeware)
- 由 Linus Toralds 開發於 1991
- 最初版本發布於 1994
----
#Linux 發行版
- Linux 內核 + 其他軟件 = 發行版
- 每個發行版著重的地方不同
- 有許多發行板可以選擇
- DistroWatch.com
- 上面版本資訊匯集網
----
#Linux 優勢
- 泛用性
- 空間小,老設備可用
- 穩定安全
- 適合伺服器
- FOSS開源
- 多項選擇
- 免費軟體生態系統
----
#總結
- Linux是操作系統
- 內核
- 軟硬體的中介
- 可以使用於伺服器or桌面
- 有各種不同的專精版本可以選擇
