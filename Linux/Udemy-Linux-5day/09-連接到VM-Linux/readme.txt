##建立連線

創建帳戶後，將為您提供有關如何連接到Linux的詳細信息。
服務器。 可能會為您提供以下一些或全部信息：
用戶名。 這也稱為帳戶，登錄名或ID。
密碼
SSH密鑰
服務器名稱或IP地址
端口號
連接協議

連接協議將是SSH（安全外殼）或telnet。 SSH和telnet提供方法
通過Internet或局域網連接到服務器。 在大多數情況下
連接協議將為SSH。 Telnet實際上已經過時了，但是您可以運行
如果您需要訪問舊版系統，請訪問它。
